Game Rules

Yahtzee is a dice game where players compete to achieve the highest score by rolling five dice and matching specific scoring combinations.

How to Play:

  1. Players take turns rolling up to five dice, aiming to match specific combinations.
  2. Each player gets three rolls per turn. After each roll, players can choose to keep or re-roll any number of dice.
  3. After rolling, players must select a score category to fill. Each category can only be used once per game.

About Yahtzee

Yahtzee is a classic dice game that has entertained players of all ages for decades. It combines strategy, luck, and skill to create a unique experience every time you play.
